## Install

* Clone repository:
```sh
git clone https://github.com/Androz2091/giveaways-bot
```

* Fill configuration (config.json):
```json
{
    "token": "your token",
    "everyoneMention": true,
    "hostedBy": true,
    "guildId": "server id"
}
```

* Install dependencies:
```sh
npm install
```

* Start:
```sh
npm start
```
